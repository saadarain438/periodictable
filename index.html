Periodic Table

The periodic table, also known as the periodic table of elements, arranges the chemical elements such as hydrogen, silicon, iron, and uranium according to their recurring properties. The number of each element corresponds to the number of protons in its nucleus (which is the same as the number of electrons orbiting that nucleus). The modern periodic table provides a useful framework for analysing chemical reactions, and is widely used in chemistry, physics and other sciences
